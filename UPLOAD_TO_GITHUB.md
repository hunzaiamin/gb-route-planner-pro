# Upload GB Route Planner Pro to GitHub

This folder is ready to upload as a new repository named `gb-route-planner-pro`.

## Upload steps

1. Open GitHub.
2. Create a new empty repository named `gb-route-planner-pro`.
3. Do not add a README, license, or `.gitignore` during repository creation.
4. Open the new repository page.
5. Choose `uploading an existing file`.
6. Upload these files and folders from this folder:
   - `index.html`
   - `road-status.csv`
   - `manifest.webmanifest`
   - `service-worker.js`
   - `README.md`
   - `UPLOAD_TO_GITHUB.md`
   - `icons/`
7. Commit the uploaded files to the `main` branch.

## Enable GitHub Pages

1. Open repository `Settings`.
2. Open `Pages`.
3. Under `Build and deployment`, select `Deploy from a branch`.
4. Select branch `main`.
5. Select folder `/root`.
6. Save.
